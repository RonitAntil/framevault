# FrameVault

A full-stack content authentication platform built with React, Supabase, and blockchain-inspired verification.

## Features
- 🔐 JWT Authentication with social login (Google, Facebook)
- 🖼️ Secure media upload with 10MB file support
- ⛓️ Blockchain timestamp verification
- 🗑️ Soft-delete with ownership-based security (RLS)
- 📱 Fully responsive design
- 🎨 Professional UI/UX with Tailwind CSS

## Tech Stack
- **Frontend:** React 18, React Router, Tailwind CSS
- **Backend:** Supabase (PostgreSQL, Auth, Storage)
- **Security:** Row Level Security (RLS), JWT tokens
- **Storage:** Supabase Buckets with CDN
